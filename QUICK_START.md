# Quick Start: Deploy in 10 Minutes

## Your Live Presentation URL
```
https://ks-bhatti.github.io/fraud-detection-presentation/
```

## 4 Simple Steps

### 1. Create Repository
- Go to https://github.com/new
- Name: `fraud-detection-presentation`
- Select **Public**
- Click **Create repository**

### 2. Upload Files
**Choose one method:**

**Option A: Web Upload (Easiest)**
- Click **Add file** → **Upload files**
- Drag drop: `index.html`, `README.md`, `.gitignore`
- Commit message: "Add fraud detection presentation"
- Click **Commit changes**

**Option B: Git Command**
```bash
git clone https://github.com/ks-bhatti/fraud-detection-presentation.git
cd fraud-detection-presentation
# Copy index.html, README.md, .gitignore here
git add .
git commit -m "Add fraud detection presentation"
git push -u origin main
```

### 3. Enable GitHub Pages
1. Go to repository Settings
2. Click **Pages** (left sidebar)
3. Under Source: Select **main** branch, **/root** folder
4. Click **Save**

### 4. Wait & Test
- Wait 1-2 minutes
- Open: https://ks-bhatti.github.io/fraud-detection-presentation/
- Use arrow keys to navigate slides

✅ Done! Your presentation is live!
